# ComfyUI-Nanguang-Translator 插件 (简化版)

一个专为ComfyUI设计的百度翻译集成插件，提供文本翻译和预处理功能。

## 功能特点

- ✅ **完全中文化界面**：所有节点界面均为中文
- ✅ **百度翻译集成**：调用百度翻译API进行高质量翻译
- ✅ **多语言支持**：支持15+种语言互译
- ✅ **文本预处理**：提供文本清洗和格式化功能
- ✅ **批量翻译**：支持多行文本批量翻译
- ✅ **专业模式**：提供专业术语和文学翻译模式
- ✅ **错误处理**：完善的错误提示和重试机制
